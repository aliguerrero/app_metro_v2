Modulo: triggers
Tabla: log_user

Sentencias SQL y funcion:

- trg_log_user_no_delete
Funcion: Bloquea eliminaciones directas sobre log_user para preservar la trazabilidad historica.
Momento: BEFORE DELETE
Definer: root@localhost
SQL:
CREATE DEFINER=root@localhost TRIGGER trg_log_user_no_delete
BEFORE DELETE ON log_user
FOR EACH ROW
SIGNAL SQLSTATE '45000'
  SET MESSAGE_TEXT = 'No se permite eliminar registros de auditoría (log_user).';

- trg_log_user_no_update
Funcion: Bloquea modificaciones directas sobre log_user para preservar la integridad de la auditoria.
Momento: BEFORE UPDATE
Definer: root@localhost
SQL:
CREATE DEFINER=root@localhost TRIGGER trg_log_user_no_update
BEFORE UPDATE ON log_user
FOR EACH ROW
SIGNAL SQLSTATE '45000'
  SET MESSAGE_TEXT = 'No se permite modificar registros de auditoría (log_user).';
